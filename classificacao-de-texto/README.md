# Classificação de Texto

## Descrição

Projeto de classificação de textos utilizando técnicas de Processamento de Linguagem Natural (NLP) e algoritmos de machine learning.

## Objetivo

Implementar e avaliar modelos de classificação de texto, explorando diferentes técnicas de pré-processamento, vetorização e algoritmos de aprendizado de máquina.

## Tecnologias Utilizadas

- **Python 3.x**
- **pandas**: Manipulação de dados
- **numpy**: Operações numéricas
- **scikit-learn**: Algoritmos de ML e vetorização
- **nltk/spacy**: Processamento de linguagem natural
- **matplotlib/seaborn**: Visualizações

## Estrutura do Notebook

1. Importação de bibliotecas
2. Carregamento e exploração dos dados
3. Pré-processamento de texto
4. Vetorização (TF-IDF, Bag of Words)
5. Treinamento de modelos
6. Avaliação de performance
7. Visualização dos resultados

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/cbdocid25/machine-learning-aplicado-II.git
```

2. Navegue até o diretório:
```bash
cd machine-learning-aplicado-II/classificacao-de-texto
```

3. Instale as dependências:
```bash
pip install pandas numpy scikit-learn matplotlib seaborn nltk
```

4. Execute o notebook:
```bash
jupyter notebook classificacao-de-texto.ipynb
```
